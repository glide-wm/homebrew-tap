# Glide-wm Tap

## How do I install these formulae?

`brew install glide-wm/tap/<formula>`

Or `brew tap glide-wm/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "glide-wm/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
